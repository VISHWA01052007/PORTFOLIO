Projects

Machine X3

machineX3 is an Industry 4.0 project developed to automate coolant monitoring in industrial machines. The system tracks Raw Coolant, Mixed Coolant, and Water tank levels using ultrasonic sensors, and measures daily consumption using flow sensors. An ESP32 collects sensor data and sends it via MQTT to a Node-RED dashboard for real-time visualization and mobile monitoring. Daily usage data is also logged automatically to Google Sheets. For industrial control, a Siemens S7-1500 PLC manages ladder logic operations, while an HMI designed in TIA Portal displays live tank levels, flow status, and alerts like low level, overfill, and no-flow conditions. OPC UA integration enables reliable communication and future readiness for SCADA/MES systems.This project enhanced my hands-on skills in IoT, industrial automation, PLC-HMI systems, and real-time monitoring solutions.
